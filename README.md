#### Small demo React app to visualize JSON files with nested object support.
- React 18
- Vite 2.9.5
- TailwindCSS 
- PostCSS 
- Autoprefixer
- React-Icons 
- JSON-BigInt

# Building

### Prerequisites:

- VSCode
- Nodejs 16 LTS or higher

### Build process:
```
npm install
npm run build
```
### To run the app without building it, use:
```
npm run dev
```
